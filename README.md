# dijkstra-algorithm
Mit hilfe dieses Programms ist es möglich den kürzesten Pfad zwischen zwei ausgewählten Städten aus der Datenbank zu finden. Der Pfad wird nach dem Dijkstra Algorithmus berechnet.
